# Imagens dos Ministérios

Pasta dedicada para armazenar as imagens dos cards de ministérios.

## Estrutura Esperada

Cada ministério necessita de uma imagem JPG com as seguintes especificações:

### Dimensões Recomendadas
- **Largura:** 400px - 600px
- **Altura:** 300px - 350px
- **Proporção:** 4:3 ou 16:9
- **Formato:** JPG (otimizado para web)
- **Tamanho:** Máximo 200KB por imagem

## Arquivos Necessários

| Arquivo | Ministério | Descrição |
|---------|-----------|-----------|
| `louvor.jpg` | Ministério de Louvor | Imagem da equipe de louvor/coro ou instrumento musical |
| `palavra.jpg` | Ministério da Palavra | Imagem de pregação/ensinamento da Palavra |
| `oracao.jpg` | Ministério de Oração | Imagem de intercessão/oração em comunidade |
| `infantil.jpg` | Ministério Infantil | Crianças em atividades/celebração |
| `acolhida.jpg` | Ministério de Acolhida | Pessoas se saudando/acolhimento |
| `social.jpg` | Ministério Social | Ação social/voluntariado |

## Como Adicionar Imagens

1. Prepare as imagens em JPG (otimizadas para web)
2. Nomeie-as conforme a tabela acima
3. Salve-as nesta pasta (`assets/images/ministerios/`)
4. As imagens serão automaticamente exibidas nos cards

## Otimização de Imagens

Use uma ferramenta online para otimizar as imagens antes de adicionar:
- [TinyJPG](https://tinyjpg.com/)
- [Compressor.io](https://compressor.io/)
- [ImageOptim](https://imageoptim.com/) (macOS)

Isto garante melhor performance e carregamento mais rápido no site.
